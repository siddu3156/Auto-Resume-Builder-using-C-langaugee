# Resume Builder using C and GTK

## Project Description

* A GUI-based Resume Builder application.
* Developed using C programming and GTK library.
* Allows users to create and save resumes in text format.
  
## Key Features

* Login and Sign-up system
* User-specific resume generation
* Simple and interactive GUI
* Automatic resume file creation

## Resume Sections

* Personal Information
* Career Objective
* Education (Degree, PUC, School)
* Skills
* Projects
* Achievements
* Languages Known
* Hobbies

## Technologies Used

* Programming Language: C
* GUI Toolkit: GTK
* Concepts Used:

  * Structures and Unions
  * File Handling
  * Event-driven Programming
    
## Working of the Project

1. User logs in or signs up.
2. Resume form is displayed.
3. User enters resume details.
4. Click on Generate Resume.
5. Resume is saved as a `.txt` file.

## Output

* Resume is saved as:

  ```
  username_resume.txt
  ```

## Advantages

* Easy to use
* Lightweight application
* Beginner-friendly project
* No database required
